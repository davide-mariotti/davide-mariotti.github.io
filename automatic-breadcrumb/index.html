Welcome to my Breadcrumb Example

A breadcrumb is a type of secondary navigation that shows the user's location in a website or web application. Here's an example of how you can create an automatic breadcrumb using JavaScript: 
            
// Get the current location
const currentLocation = window.location.pathname;

// Split the location into an array of levels
const locationArray = currentLocation.split('/');

// Create the breadcrumb container
const breadcrumb = document.createElement('div');
breadcrumb.classList.add('breadcrumb');

// Iterate through the location levels and create breadcrumb links
locationArray.forEach((level, index) => {
// Skip the first level (it's usually just a forward slash)
if (index === 0) return;

// Create the breadcrumb link
const link = document.createElement('a');
link.href = locationArray.slice(0, index + 1).join('/');
link.textContent = level;

// Append the link to the breadcrumb container
breadcrumb.appendChild(link);
});

// Append the breadcrumb container to the body
document.body.appendChild(breadcrumb);
In this example, the currentLocation variable is set to the current URL using window.location.pathname. This is then split into an array of levels using the split() method. A container is created to hold the breadcrumb element and each level is iterated over to create a link that corresponds to the level. The link is then added to the container and then the container is appended to the body of the HTML. You can also style the breadcrumb with CSS according to your preference, for example: 
            
.breadcrumb {
    display: flex;
    align-items: center;
    padding: 0.5em;
    background-color: #f5f5f5;
}
.breadcrumb a {
    color: #333;
    text-decoration: none;
    margin-right: 0.5em;
}
This is a basic example of how you can create an automatic breadcrumb, but you can customize it to suit your specific needs. You can also use libraries like Bootstrap, Foundation, Material-UI, etc to create breadcrumbs.
